Accessing the Path

Under the hood, Ruxt.rs uses the Actix web framework to handle requests. This means that you can access the path of the current request using the Path struct from Actix.

Here's an example of how you can access the path of the current request:

use actix_web::web::Path;
use serde::Deserialize;

#[derive(Deserialize)]
struct TestPath {
    id: i32
}

async fn index(path: Path<TestPath>) -> String {
    format!("The id is: {}", path.id)
}

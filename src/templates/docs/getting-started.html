What is Ruxt.rs?

Ruxt.rs is an Actix-based Rust framework for building HTMX sites. It allows you to build dynamic web applications using Rust and HTMX, a modern approach to building web applications that leverages the power of server-side rendering and the simplicity of HTML.

The goal of Ruxt.rs is to provide a simple, easy-to-use framework for building web applications that are fast, secure, and maintainable. It is designed to be lightweight and flexible, allowing you to build web applications that meet your specific needs.

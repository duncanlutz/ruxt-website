Creating Routes

Ruxt.rs uses a file-based routing system. This means that the file structure of your project will determine the routes of your application.

Each folder in the src/pages directory maps to a route segment in your application.

They follow the following rules:

  • An index.rs file in a folder will be the route for that folder. (e.g. src/pages/about/index.rs will be the route /about)
  • A page_name.rs file in a folder will be the route for that file. (e.g. src/pages/about.rs will be the route /about)
  • Files that begin with two underscores (e.g. src/pages/__layout.rs) denote a dynamic path segment. (e.g. /user/{id}) You can learn how to access these values here.

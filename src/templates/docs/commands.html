{{ codeblock::codeblock(code="cargo run", id="run-project") }}

You may also want to install the cargo-watch crate to automatically reload the server when you make changes. You can install it by running:

{{ codeblock::codeblock(code="cargo install cargo-watch", id="install-watch") }}
